💻 51 Commandes Linux de base à connaître ! 🚀

📌 Introduction

Cette vidéo présente 51 commandes Linux essentielles pour les débutants et les utilisateurs avancés.

🗓️ Gestion du temps

  • date - Affiche la date actuelle.
  • date +%d-%m-%Y %H:%M:%S - Personnalise l'affichage.

📖 Obtenir de l'aide

  • man [commande] - Affiche le manuel d’une commande.
  • q - Quitter le manuel.

👤 Gestion des utilisateurs

  • w / who - Voir les utilisateurs connectés.
  • id - Affiche l'ID utilisateur et groupe.

📂 Navigation dans les fichiers

  • pwd - Affiche le répertoire actuel.
  • ls -la - Liste les fichiers, y compris les fichiers cachés.

🔑 Gestion des permissions

  • chmod o-x [fichier] - Supprime l'exécution pour les autres.

🔗 Liens et raccourcis

  • ln -s [fichier] [lien] - Crée un lien symbolique.
  • unlink [lien] - Supprime un raccourci.

📁 Manipulation de répertoires

  • mkdir [nom] - Crée un dossier.
  • cd [nom] - Se déplacer dans un dossier.

📝 Gestion des fichiers

  • echo "texte" > fichier.txt - Écrit du texte dans un fichier.
  • cat [fichier] - Affiche le contenu d’un fichier.

🔎 Recherche et tri

  • grep "mot" [fichier] - Recherche un mot dans un fichier.
  • find /home -name "*.txt" - Recherche des fichiers par nom.

📤 Copie et suppression

  • cp [source] [destination] - Copie un fichier.
  • mv [source] [destination] - Déplace ou renomme un fichier.
  • rm [fichier] - Supprime un fichier.

📊 Informations système

  • uname -a - Affiche les infos du système.
  • top - Affiche les processus en cours.

🌍 Réseau

  • ifconfig - Affiche les interfaces réseau.
  • ping google.com - Teste la connexion Internet.

🛠️ Administration

  • sudo [commande] - Exécute une commande en admin.
  • passwd - Change un mot de passe.

🗄️ Gestion du stockage

  • df -h - Affiche l’espace disque disponible.
  • du -sh [répertoire] - Affiche la taille d’un dossier.

🗜️ Compression

  • zip -r archive.zip dossier - Compresse un dossier.
  • unzip archive.zip - Décompresse une archive.

🛑 Arrêter et redémarrer

  • shutdown -h now - Éteint immédiatement.
  • reboot - Redémarre le système.

📢 Conclusion

Ces commandes sont essentielles pour maîtriser Linux. 🐧 Prêt à les tester ? 🚀
