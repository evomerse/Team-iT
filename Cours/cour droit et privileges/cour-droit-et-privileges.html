🔥 Formation : Création et gestion d’un serveur web

📢 Cette formation couvre l’installation et la gestion d’un serveur web Apache sur une distribution Linux Ubuntu pour héberger des sites en local.

🎯 Plan de la formation :

  • 1️⃣ Installation d’Apache – Configuration et mise en place du serveur web.
  • 2️⃣ Hébergement d’un site web – Configuration des fichiers nécessaires.
  • 3️⃣ Création d’un nom de domaine – Association d’un domaine personnalisé au serveur.

🛠️ Installation d’Apache :

  • ✅ Vérification de la présence d’Apache.
  • ✅ Mise à jour du système avec sudo apt update.
  • ✅ Installation avec sudo apt install apache2.
  • ✅ Vérification du statut du service avec systemctl status apache2.

🌐 Test du serveur :

  • 💻 Récupération de l’adresse IP avec ifconfig.
  • 🌍 Accès au serveur via le navigateur en entrant l’adresse IP.

🎬 Prochaine étape :

🚀 Hébergement d’un site web sur Apache.
